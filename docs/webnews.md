
## April 2026

QuickView version 2.5.0 is released, providing long-desired features like image download, symmetric logarithmic scale for colorbars, generalized support for [arbitrarily dimensioned arrays](/guides/simulation_data#nd-vars), and substantially improved performance for [SCREAM output](/gallery/index#scream).
