
## April 2026

QuickView version 2.5.1 is released, providing long-desired features like [image download](/guides/quickview/miscellaneous#save-vis), [symmetric logarithmic scale](/guides/quickview/individual_views#linear-and-log-scales) for colorbars, generalized support for [arbitrarily dimensioned arrays](/guides/simulation_data#nd-vars), and substantially improved performance for [SCREAM output](/gallery/index#scream).
